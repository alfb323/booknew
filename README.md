# Libros Seguros 📚

Una plataforma web minimalista, rápida y accesible para encontrar plataformas de lectura digital libres de virus, estafas o piratería. El objetivo de este sitio es centralizar bibliotecas de dominio público y préstamo digital legal.

## ✨ Características

- **Filtrado instantáneo:** Combinación de buscador por texto y botones de categoría en tiempo real (sin recargar la página).
- **Rendimiento optimizado:** Filtrado mediante atributos `data-tags` nativos de HTML y optimización de escritura (*debounce*) en JavaScript.
- **Diseño Editorial Clean:** Paleta de colores inspirada en papel y lectura nocturna, completamente adaptada a dispositivos móviles.
- **Accesibilidad Web:** Estructura semántica avanzada y atributos ARIA para soporte completo de lectores de pantalla.
